# Examples for this fork

## Safari PWA + Voice + CrewAI Starter
Ready-to-run example for iOS/Safari installable PWA with voice input and backend agent endpoint.

- Path: `examples/pwa-safari-voice-starter/`
- Entry point: `examples/pwa-safari-voice-starter/README.md`

## Includes
- `manifest.webmanifest` + iOS meta tags
- Service Worker with offline fallback
- Voice command flow (Web Speech API)
- Offline queue with retry/backoff
- FastAPI backend endpoint at `/api/agent`
